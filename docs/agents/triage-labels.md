# Triage Labels

The skills speak in terms of canonical triage roles. This repo uses only the implementation-readiness labels below.

| Label in mattpocock/skills | Label in our tracker | Meaning                                 |
| -------------------------- | -------------------- | --------------------------------------- |
| `ready-for-agent`          | `ready-for-agent`    | Fully specified, ready for an AFK agent |
| `ready-for-human`          | `ready-for-human`    | Requires human implementation           |

## Unused triage roles

This repo does not use these canonical triage labels:

- `needs-triage`
- `needs-info`
- `wontfix`

When a skill mentions a role, use the corresponding label string from the table above. Do not create or apply the unused triage labels unless the repo's workflow changes.
